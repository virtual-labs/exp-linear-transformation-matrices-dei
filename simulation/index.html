Linear Transformations

NOTE: Consider the vector spaces R, R2 over R

T:V → V, where V is a vector space over R

T(x) = 0 ; x ∈ V

T is linear

T:R2 → R2

T(x) = (1, 1); x ∈ R

T is not linear

T:R2 → R2

T(x, y) = (x, 0); x, y ∈ R

T is linear

T:R → R2

T(x) = (x, 3x); x ∈ R

T is linear

T:R → R2

T(x) = (x, 3x+1); x ∈ R

T is not linear

[∵ T(0) ≠ 0
T(1+2) ≠ T(1)+T(2)]

T:R → R2

T(x) = (x, x2); x ∈ R

T is not linear

[∵ T(0) ≠ 0
